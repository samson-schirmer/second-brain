# Agent Development Lifecycle (ADLC)

The six-phase framework governing all agent work at the GenAI COE.

## Phases

1. **Intake & Prioritization** — Request captured via MS Forms → Workfront
2. **Design & Scoping** — Requirements defined, stakeholders aligned
3. **Build & Test** — Agent built, tested for quality and brand compliance
4. **Deploy & Release** — Agent released to users
5. **Monitor** — Performance tracked; user feedback collected
6. **Govern** — Brand, compliance, and quality gates maintained

## Key Insight

Agents reach "functionally complete" quickly, but "reliable and trustworthy" takes continuous iteration. This distinction shapes timelines and resource expectations — stakeholders expect a deployed agent to be done, but the real work is in the Monitor → Govern loop.

## Tools

- **Intake:** Microsoft Forms (automated)
- **Tracking:** Workfront
- **Build:** Writer.ai platform
- **Governance:** Brand Hub guidelines
